Digitial Profile is Protected

A cross-browser extension that encompasses the necessary features to best protect individuals against common cyber security risks whilst surfing the web.

The privacy and confidentiality of personal data are of utmost importance to organisations and individuals alike. However, some of this data is collected by web browsers without the knowledge of the users. These concerns are amplified by the proliferation of smart devices, capable of connecting to the internet. Web browsers are available across many devices and platforms. Ensuring the usersâ€™ security and privacy is a significant challenge. Zunna is designed to protect personal data.


The main features of Zunna are:

  • Clickjacking Protection: The extension can detect and highlight hidden inline frame features in a URL and Web Page.
  • Cookie File Protection: The extension can manage and delete appropriate cookie files in a URL and Web Page.
  • Cross-Site Scripting Protection: The extension can recognize and remove Cross-Site Scripting tags in a URL and Web Page.
  • Password suggestions: Users receive a prompt for a more secure and memorable password when visiting sign up pages.
  • Safe Website Checking: The extension can recognize when a site is safe or not using the HTTP protocol and comparing URLs to datasets of websites known to be dangerous, cautioning the user appropriately.
  • Tracking Protection: The extension can recognize and remove tracking components in a URL and Web Page.
  • Advertisement Blocking: The extension can recognize and remove advertisement elements on a web page.
